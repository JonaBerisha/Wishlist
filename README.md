# Wishlist

Projektet er en webbaseret aplikation, hvor man som bruger kan oprette ønskeseddler og tilføjer ønsker med navn pris og link til produktet. Herefter kan brugeren selv søge og slette ønsker, og kan sortere ønskerne efter pris.

## Features
Opret bruger
Login
Opret ønskeseddel
Tilføje ønkser
Slet ønsker
Søge efter ønsker
Sorter ønsker

## Technologies
Java
Spring Boot
Thymeleaf
SQL(jbdc Templates)
Azure
Github Actions

## Instalation
Java 21
Maven
MySQL

## Usage
1. Opret en bruger
2. Login med email og password
3. Opret en ønskeseddel
4. Tilføj ønsker til ønskeseddel
5. Brug søgningen til at finde ønsker
6. Sorter ønsker efter pris
7. Slet øsnke hvis nødvendigt
